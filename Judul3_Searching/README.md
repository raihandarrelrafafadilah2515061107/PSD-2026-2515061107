Program Sequential Search Ukuran Sepatu Mahasiswa

Program ini merupakan implementasi metode Sequential Search dalam kehidupan sehari-hari untuk mencari jumlah mahasiswa yang memiliki ukuran sepatu tertentu. Sequential Search bekerja dengan cara memeriksa data satu per satu secara berurutan dari awal hingga akhir. Jika ukuran sepatu yang dicari ditemukan, maka program akan menghitung berapa kali ukuran tersebut muncul di dalam daftar data.

Pada program ini, pengguna diminta memasukkan ukuran sepatu yang ingin dicari. Program kemudian akan memeriksa seluruh data ukuran sepatu mahasiswa secara berurutan menggunakan perulangan. Setelah pencarian selesai, program akan menampilkan apakah ukuran sepatu tersebut ditemukan dan berapa jumlah kemunculannya di dalam daftar data.
